# Description
Molehill Chemical Calculator is an application that calculates the amount of reagents to
use when mixing up chemical solutions.

It is intended for use by chemists and other scientists who have enough knowledge and experience to use it safely.

Molehill Chemical Calculator  was written and copyrighted by Chris Baba in 2024.
It is open source software licensed under GPLv3 or later.

# Installing
Molehill Chemical Calculator is currently available for Windows 11 and some Linux distributions.

**Windows 11:**
An installer/remover is available for Windows 11 in the `molehill_win-installer` directory above.  Click on that directory then click on  the `Molehill_Installer.exe` file.  Then download it by clicking on the download button on the right.  This single file contains everything that is needed to Install Molehill Chemical Calculator on Windows 11.  

Please use the `Molehill_Installer.exe` file to install to all of your Windows 11 Computers and share it with your friends and coworkers.


**Linux**
Right now there is no general Linux installer due to the various requirements needed for each Linux distribution.  The Linux end user should first look for Molehill Chemical Calculator in their distribution's package repository and use that if available.

Distribution package maintainers, and skilled Linux end users can try to use the Molehill Chemical Calculator applications executable which is called `chemcalc` and is in the `molehill-linux-installer` subdirectory.  The dependicies needed can be found by using `ldd chemcalc` from your command line. or looking at `ldd-dependencies.txt` in the Molehill github source code.
